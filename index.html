i66.org

One-command shortcuts for cloud and Linux infrastructure. Fetch and execute AWS, Azure, GCP, Kubernetes, Docker, and system commands without memorising flags or writing scripts.

bash — i66
$ i66 aws/ec2/list
# fetches http://i66.org/aws/ec2/list → runs via bash
INSTANCE-ID NAME TYPE STATE IP
i-0abc123def456789 web-prod-01 t3.medium running 10.0.1.4
i-0def789abc012345 db-primary r6g.large running 10.0.2.8

Install i66

go build -o i66 i66.org.go
./i66 <command>

# or grab a pre-built binary

No install (curl)

bash <(curl -s http://i66.org/sys/disk)

curl -s http://i66.org/sys/memory | bash
🎯

Target a host (env vars)

DOMAIN=example.com i66 net/dig/a
HOST=8.8.8.8       i66 net/ping
IP=1.2.3.4         i66 net/dig/ptr
URL=https://…      i66 net/headers
🐧
Requires Linux or a Linux-compatible shell. Commands use bash, systemctl, journalctl, ss, ip, and other Linux/GNU tools — they will not work in Windows PowerShell or CMD. On Windows, use WSL2 (Windows Subsystem for Linux) or SSH into a Linux server. Required tools are listed as badges on each section header.
⚠  Commands are fetched over HTTP and executed locally with full user privileges. Only run against servers you control. Click any command/path to copy the i66 invocation. Click the curl badge to copy a no-install bash one-liner.
